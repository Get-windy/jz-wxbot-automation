## 1. 项目概述

jz-wxbot 是一个基于 pyautogui 和 win32gui 的微信自动化机器人，支持个人微信和企业微信双开。

### 1.1 核心技术栈
- **GUI自动化**: pywinauto, pyautogui
- **消息处理**: 自定义消息队列
- **桥接服务**: OpenClaw集成
- **消息协议**: WebSocket, HTTP

### 1.2 项目结构
```
jz-wxbot-automation/
├── bridge/              # 桥接服务层
│   ├── bridge_service.py
│   ├── message_listener.py
│   └── openclaw_client.py
```
