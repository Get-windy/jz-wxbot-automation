## 📋 错误代码速查表

### 微信核心错误 (WX)

| 错误码 | 错误名称 | 症状 | 快速解决 |
|--------|---------|------|---------|
| **WX-R-001** | 窗口未找到 | 提示"未找到微信窗口" | 1. 启动微信 2. 恢复窗口(非最小化) |
| **WX-R-002** | 进程不存在 | 进程列表无WeChat.exe | 启动微信客户端 |
| **WX-R-003** | 窗口无响应 | 操作卡死/超时 | 重启微信 |
| **WX-S-001** | 剪贴板失败 | 复制粘贴不生效 | 1. 重试 2. 清理剪贴板 3. 重启应用 |
| **WX-P-001** | 权限不足 | 窗口操作被拒绝 | 以管理员身份运行 |

### 消息错误 (MSG)

| 错误码 | 错误名称 | 症状 | 快速解决 |
|--------|---------|------|---------|
| **MSG-R-001** | 发送失败 | 消息未发出 | 1. 检查联系人名称 2. 确认窗口就绪 |
| **MSG-R-002** | 发送超时 | 等待时间过长 | 1. 增加timeout配置 2. 检查网络 |
| **MSG-S-001** | 输入框未找到 | 无法定位输入区域 | 1. 确认微信前台 2. 点击聊天窗口 |

### 窗口错误 (WIN)

| 错误码 | 错误名称 | 症状 | 快速解决 |
|--------|---------|------|---------|
| **WIN-R-001** | 句柄无效 | 操作失败 | 重新初始化发送器 |
| **WIN-R-002** | 窗口已关闭 | 窗口消失 | 重新打开微信窗口 |

### 自动化错误 (AUTO)

| 错误码 | 错误名称 | 症状 | 快速解决 |
|--------|---------|------|---------|
| **AUTO-R-001** | 操作失败 | 自动化步骤出错 | 1. 检查操作序列 2. 查看日志 |
| **AUTO-T-001** | 执行超时 | 操作未完成 | 增加超时时间配置 |

### MCP服务错误 (MCP)

## 🔧 常见问题快速修复

### 1. 微信窗口未找到 (WX-R-001)

```powershell
# 检查微信进程
Get-Process | Where-Object {$_.Name -like "*WeChat*" -or $_.Name -like "*WXWork*"}

# 启动微信
Start-Process "C:\Program Files\Tencent\WeChat\WeChat.exe"
```

```python
# Python 重初始化
sender = WeChatSenderV3()
sender.reinitialize()
```

### 2. 消息发送失败 (MSG-R-001)

```python
# 带重试的发送
def send_with_retry(sender, target, msg, retries=5):
    for i in range(retries):
        try:
            result = sender.send_message(target, msg)
            if result:
                return True
        except:
            time.sleep(2)
    return False
```

### 3. MCP服务连接失败 (MCP-N-001)

```powershell
# 检查端口
netstat -ano | findstr :8080

# 重启服务
net stop jz-wxbot-mcp
net start jz-wxbot-mcp
```

### 4. 剪贴板操作失败 (WX-S-001)

```python
# 安全剪贴板操作
import pyperclip
import time

def safe_copy(text, retries=3):
    for _ in range(retries):
        try:
            pyperclip.copy(text)
            if pyperclip.paste() == text:
                return True
        except:
            time.sleep(0.5)
    return False
```
