## 1️⃣ 身份认证安全测试

### TC-WX-SEC-AUTH-001: 微信扫码登录安全
**测试目的**: 验证微信扫码登录流程安全性
**测试步骤**:
1. 生成登录二维码
2. 验证二维码有效期（应≤5分钟）
3. 扫码后验证回调URL安全性
4. 验证Token生成使用强随机算法
5. 验证Session绑定设备指纹

**预期结果**:
```yaml
二维码有效期: ≤300秒
Token熵值: ≥128位
Session绑定: 设备指纹+IP
重放攻击防护: 单次使用二维码
```

**测试代码**:
```python
def test_qr_code_security():
    # 生成二维码
    qr_data = generate_login_qr()
    
    # 验证有效期
    assert qr_data['expires_in'] <= 300
    
    # 验证Token强度
    token_entropy = calculate_entropy(qr_data['token'])
    assert token_entropy >= 128
    
    # 验证单次使用
    scan_qr(qr_data['token'])
    with pytest.raises(InvalidTokenError):
        scan_qr(qr_data['token'])  # 重复使用应失败
```
**优先级**: P0

```yaml
错误信息: 不暴露SQL细节
ORM防护: 启用
```

**测试代码**:
```python
def test_sql_injection_protection():
    malicious_inputs = [
        "' OR '1'='1",
        "; DROP TABLE users; --",
        "' UNION SELECT * FROM passwords --"
    ]
    
    for payload in malicious_inputs:
        response = send_message(payload)
        assert response.status_code != 500
        assert "sql" not in response.text.lower()
        assert "database" not in response.text.lower()
```
